CSS

  1. the difference between the HTML Source Code and what we see on Screen is the Data Structure .
  2. Tree Diagram Shows which elements are nested into each other , and looks like a Map for how the webpage will look like.
  3. Tree structure consists of some elements called Nodes and connections between them called Branches . Every tree has a Root and the root of the HTML Page is the HTML element.
  4. Rules for Tree Structures :
    1. A Tree has a Root Node.
    2. Each Node can have Branches that lead to other nodes(its Children).
    3. Each Node has only one Parent (Except the root).
  5. CSS Syntax Ruleset:
    1. Selector: which element will the ruleset apply to ?
    2. Decleration Block how will the ruleset Modify these elements ?
    3. in decleration block : "color : green;" color is the property and green is the value .
  6. a Class is a group of things with the same characteristics.
  7. to make a class just get it a name and start using it. For Example : "-p class="introduction"-". Also called in style using a Dot (.) like ".introduction".
  8. a Class is used when there're several elements that need the same style.
  9. an ID is used when there's only one element to apply to. and called in style using "#".
  10. the span is inline container used to Markup a part of text or a part of a document.
  11. the compining selector is the compine of two selectors in one style like "li em" as we only choose em in li to apply our characteristics to and called "The Descendant selector"
  12. we can compine two different classes by seperating between them by a space like class="important done".
  13. The term "cascading" in Cascading Style Sheets refers to the way that style properties "cascade" down the DOM tree, starting at the top. A style applied to the body element will affect the entire document. A style applied to a lower-level element will affect that element and all its descendants.
  14. there's many sizing options in CSS like em and it uses the same font size of the element or px for Pixel and it's 1/96 inch and Pt for the font size .
  15. we can Layout the blocks by using padding , margin and border also float is used to change the position of the block.
